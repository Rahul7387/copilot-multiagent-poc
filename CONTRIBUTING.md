# Contributing

Thank you for your interest in contributing to this project!

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/add-finance-agent`
3. Make your changes
4. Commit with a clear message: `git commit -m "Add Finance agent with expense and budget topics"`
5. Push to your fork: `git push origin feature/add-finance-agent`
6. Open a Pull Request

## Ideas for contribution

- Add new specialist agents (Finance, Facilities, Legal, Payroll)
- Add Power Automate flow examples for ticket creation or data lookup
- Add Dataverse knowledge source configuration
- Improve routing instructions for edge cases
- Add adaptive card examples for richer responses
- Add Teams channel deployment guide
- Translate documentation to other languages

## Guidelines

- Keep agent descriptions specific and domain-scoped
- Include trigger phrases for every new topic
- Document the conversation flow for complex topics
- Test routing before submitting changes
- Follow sentence case in all documentation

## Reporting issues

Open a GitHub Issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Copilot Studio environment details (if relevant)
