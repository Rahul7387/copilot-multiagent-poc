# IT Helpdesk Agent — topics

## Topic 1: Password reset

Trigger phrases:
- forgot password
- account locked
- reset my password
- cannot log in
- password expired
- unlock account

Conversation flow:
1. Ask which system (Windows / Microsoft 365 / VPN / Other)
2. Guide to self-service reset portal or walk through steps
3. Confirm access restored
4. If unable to resolve, raise a ticket automatically

## Topic 4: Raise IT ticket

Trigger phrases:
- log a ticket
- raise IT issue
- create support request
- IT complaint
- escalate my issue

Conversation flow:
1. Collect: Issue description, urgency level (Low / Medium / High)
2. Collect: Asset or device involved (if applicable)
3. Confirm ticket creation with ticket number
4. Set SLA expectation (High priority: 4 hours response)
